    ең–еғҸеҲҶеүІз¶“е…ёз¶Іи·Ҝ U-Net

    еҪұеғҸеҲҶеүІеҸҰеӨ–дёҖеҖӢе…·жңүд»ЈиЎЁжҖ§зҡ„ CNN з¶“е…ёжЁЎеһӢжҳҜ U-netпјҢд»–жҳҜиў«з”ЁдҫҶиҷ•зҗҶйҶ«зҷӮеҪұеғҸдёҰзҷјиЎЁж–ј MICCAI 2015гҖӮU-Net иҲҮ FCN еҗҢжЁЈйғҪжҳҜжІ’жңүе…ЁйҖЈжҺҘеұӨеҸӘжңүжҚІз©ҚеұӨпјҢеҗҢжЁЈжҳҜйҖҗеұӨиһҚеҗҲдёҚеҗҢе°әеәҰзү№еҫөең–пјҢдҪҶU-NetжӣҙеҠ еғҸдёҖеҖӢйқһеёёиҰҸж•ҙзҡ„Encoder-Decoderзөҗж§ӢгҖӮдёҰдё”йғҪжҳҜжҺЎз”Ё deconvolution е°ҮдҪҺи§ЈжһҗеәҰеҪұеғҸйӮ„еҺҹй«ҳи§ЈжһҗеәҰпјҢдҪҶжҳҜж•ҙй«”зҡ„ Upsampling йҒҺзЁӢе…©иҖ…жҳҜдёҚеӨӘдёҖжЁЈзҡ„гҖӮ

    U-Net жһ¶ж§Ӣи§Јжһҗ

    еӣ зӮәз¶Іи·Ҝжһ¶ж§Ӣй•·еҫ—еҫҲеғҸ U еһӢеӣ жӯӨзЁұдҪң U-netпјҢеҸҲеӣ зӮәеҜҰдҪңжҷӮжңғжӢҶеҲҶ Encoder-Decoder еӣ жӯӨжҲ‘еҖ‘еҸҲзЁұе®ғжҳҜеҖӢ AutoEncoder зҡ„зөҗж§ӢгҖӮи«–ж–ҮдёӯжҸҗеҮәпјҢе·ҰеҚҠзҡ„з·ЁзўјйғЁеҲҶеҸ« Contracting PathпјҢз”Ёж–јиҗғеҸ–йҮҚиҰҒиЁҠжҒҜгҖӮеҸійӮҠзҡ„и§ЈзўјйғЁеҲҶеҸ« Expanding PathпјҢз”Ёж–јзІҫзўәе®ҡдҪҚйӮ„еҺҹи§ЈжһҗеәҰгҖӮ

    ж•ҙй«”жһ¶ж§ӢеҰӮдёӢең–жүҖзӨәз•¶жҲ‘еҖ‘ијёе…ҘдёҖејө 572*572 зҡ„еҪұеғҸпјҢи—Қз·ҡз®ӯй ӯд»ЈиЎЁйҖІиЎҢдәҶ 3*3 зҡ„еҚ·з©ҚдёҰз¶“йҒҺ relu еҮҪж•ёпјҢжүҖд»ҘдёҖй–Ӣе§Ӣз¶“йҒҺе…©еұӨеҚ·ж©ҹеҫҢеҫ—еҲ° 64 ејөзҡ„ 568*568 зү№еҫөең–гҖӮиҖҢзҙ…иүІзҡ„з®ӯй ӯд»ЈиЎЁз¶“йҒҺдёҖеҖӢ 2*2 зҡ„ polingпјҢжӯӨжӯҘй©ҹжңғе°ҮеҪұеғҸеӨ§е°ҸжёӣеҚҠгҖӮеңЁе·ҰеҚҠйӮҠзёҪе…ұйҖІиЎҢдәҶеӣӣзө„ downsampling ж“ҚдҪңеҫҢеҫ—еҲ°зҡ„жңҖдҪҺи§ЈжһҗеәҰзӮә 32*32гҖӮжҺҘдёӢдҫҶдёҖжЁЈжңғжңүеӣӣзө„ Upsampling йӮ„еҺҹеҲ°й«ҳи§ЈжһҗеәҰпјҢеҖјеҫ—дёҖжҸҗзҡ„жҳҜйҖҷиЈЎзҡ„ Upsampling йҒҺзЁӢиҲҮ FCN дёҚдёҖжЁЈгҖӮйӣ–з„¶йғҪжҳҜжҺЎз”Ё deconvolution дҪҶжҳҜиҷ•зҗҶ feature maps зҡ„ж–№жі•дёҚдёҖжЁЈгҖӮйҰ–е…Ҳе°Ү 28*28 еҪұеғҸйҖІиЎҢеҸҚеҚ·з©Қеҫ—еҲ° 56*56пјҢеҸҜжҳҜжҲ‘еҖ‘зҷјзҸҫд»–зҡ„и·іиҖҖйҖЈжҺҘе·ҰйӮҠжҳҜ 64*64 еӨ§е°ҸгҖӮжӯӨжҷӮзҡ„еҒҡжі•жңғе°Үе·ҰйӮҠ 64*64 зҡ„еҪұеғҸеҸ–дёӯй–“з¬ҰеҗҲеҸійӮҠ 5656 зҡ„еӨ§е°ҸйҖІдёӢиЎҢжӢјжҺҘи®ҠжҲҗ 1024 з¶ӯгҖӮжӯӨеӢ•дҪңдёҖжЁЈеңЁ Upsampling йҖІиЎҢеӣӣзө„жңҖеҫҢеҫ—еҲ° 392*392 зҡ„еӨ§е°ҸгҖӮжҺҘи‘—еҶҚеҒҡе…©зө„ 33 зҡ„еҚ·з©ҚжңҖеҫҢеҶҚйҖҸйҒҺдёҖеҖӢ 1*1 еҚ·з©Қеҫ—еҲ° 388*388 йҖҡйҒ“зӮә 2 зҡ„еҪұеғҸеҲҶеҲҘжҳҜзҙ°иғһиҲҮйқһзҙ°иғһзҡ„ж©ҹзҺҮең–гҖӮ

    дёҖеҖӢжӯЈиҰҸзҡ„ U-net жһ¶ж§ӢжҳҜз”ұе·ҰйӮҠ contracting path е’ҢеҸійӮҠ expanding path зө„еҗҲиҖҢжҲҗгҖӮе…¶дёӯ contracting path жҳҜз”ұеҚ·з©ҚиҲҮжңҖеӨ§жұ еҢ–жүҖзө„жҲҗпјҢеӣ жӯӨи§ЈжһҗеәҰжңғйҖҗжјёйҷҚдҪҺгҖӮиҖҢ expanding path жҳҜз”ұеҸҚеҚ·з©Қ(и«–ж–ҮзЁұ Up-conv) иҲҮе·ҰйӮҠзҡ„й«ҳи§ЈжһҗеәҰеҪұеғҸ channel-wise зө„еҗҲгҖӮ

    FCN иҲҮ U-net е·®з•°

    еӣ жӯӨжҲ‘еҖ‘йҖҷиЈЎеҸҜд»ҘеҒҡдёҖеҖӢзөұж•ҙжҜ”ијғ FCN иҲҮ U-net зҡ„е·®з•°гҖӮFCN еңЁ Upsampling жҳҜйҖҸйҒҺ element-wise иҲҮеүҚдёҖеұӨзҡ„зү№еҫөең–зӣёеҠ гҖӮиҖҢ U-net жҳҜжҺЎз”Ёе·ҰйӮҠй«ҳи§ЈжһҗеәҰиіҮиЁҠ channel-wise concatenation зө„еҗҲиҖҢжҲҗгҖӮ

    FCNдёӯдҪҝз”Ёзҡ„жҳҜйҖҗй»һзӣёеҠ зҡ„ж–№ејҸ add()пјҢиҖҢU-NetдёӯжҳҜжҢү channel з¶ӯеәҰжӢјжҺҘеңЁдёҖиө· concat()пјҢзӣҙжҺҘе°ҮйҖҡйҒ“ж•ёзҝ»еҖҚпјҢеҪўжҲҗжӣҙеҺҡзҡ„зү№еҫөең–гҖӮ

    еҪұеғҸеүҚиҷ•зҗҶпјҡ Overlap-Tile Strategy ж–№жі•

    еҸҰеӨ–еңЁ U-net жһ¶ж§ӢдёӯеҚ·з©ҚеұӨдёҰз„ЎдҪҝз”Ё paddingпјҢеӣ жӯӨжҜҸйҒҺдёҖеұӨеҚ·з©Қзү№еҫөең–зҡ„еӨ§е°ҸйғҪжңғйҖҗжјёзё®е°ҸгҖӮијёе…ҘжҳҜ 572Г—572пјҢжңҖзөӮијёеҮәжҳҜ 388Г—388гҖӮзӮәдәҶжҺ§еҲ¶и§ЈжһҗеәҰе’ҢжІ’жңү padding жүҖйҖ зЁӢзҡ„еҪұжғіи«–ж–ҮдёӯжҸҗеҮә Overlap-Tile Strategy ж–№жі•гҖӮеҒҮиЁӯжҲ‘иҰҒй җжё¬ең–дёӯй»ғиүІжЎҶжЎҶзҡ„ең–еғҸеҲҶеүІе…§е®№пјҢе°ұд»Ҙй»ғиүІзӮәдёӯеҝғйҮҚж–°иЈҒеҲҮдёҖеҖӢијғеӨ§зҡ„и—ҚиүІжЎҶжЎҶзҡ„еҪұеғҸдҪңзӮәијёе…ҘпјҢз¶“йҒҺжҜҸдёҖеұӨдёҚж–·зҡ„еҚ·з©ҚжңҖеҫҢеҮәеҮәзҡ„зөҗжһңеүӣеҘҪжҳҜй»ғиүІзҡ„еҚҖеҹҹгҖӮ

    еңЁеҜҰдҪңйҒҺзЁӢдёӯеҚ·з©ҚеҸҜд»ҘжҺЎз”Ё padding=same жҺ§еҲ¶зү№еҫөең–еӨ§е°ҸгҖӮдёҰжҺЎз”Ё pooling ж©ҹеҲ¶е°Үзү№еҫөең–зё®е°ҸгҖӮйҖҷжЁЈе°ұеҸҜд»Ҙи§Јжұәијёе…ҘијёеҮәеҪұеғҸеӨ§е°ҸдёҚдёҖзҡ„е•ҸйЎҢгҖӮ

    еҜҰй©—зөҗжһң

    и«–ж–ҮиЈЎжҺЎз”Ёе…©еҖӢе…¬й–ӢйҶ«зҷӮеҪұеғҸиіҮж–ҷйӣҶйҖІиЎҢеҜҰй©—гҖӮжҲ‘еҖ‘еҸҜд»ҘзҷјзҸҫ U-net еңЁ IOU зҡ„и©•дј°дёҠйғҪзҚІеҫ—зӣёз•¶еҘҪзҡ„жҲҗзёҫгҖӮ

    QA

    • й—ңж–ј CNN иӢҘжҜҸдёҖеұӨеҚ·з©Қзҡ„ kernel size йғҪзӯүж–јеҺҹең– H*W жҳҜдёҚжҳҜе°ұзӯүеҗҢж–је…ЁйҖЈжҺҘеұӨз¶Іи·Ҝпјҹ

    • CNNдёӯзҡ„convolutionе–®е…ғжҜҸж¬ЎеҸӘй—ңжіЁй„°еҹҹkernel size зҡ„еҚҖеҹҹпјҢе°ұз®—еҫҢжңҹж„ҹеҸ—йҮҺи¶ҠдҫҶи¶ҠеӨ§пјҢзөӮз©¶йӮ„жҳҜеұҖйғЁеҚҖеҹҹзҡ„йҒӢз®—гҖӮйҖҷдёҖй»һи©ІеҰӮдҪ•и§ЈжұәпјҹжҺЎз”ЁжіЁж„ҸеҠӣж©ҹеҲ¶еҸҜд»Ҙи§ЈжұәйҖҷеҖӢе•ҸйЎҢе—Һ

    • FCN жңүдҪҝз”ЁеҲ° Unpooling жҠҖе·§е—ҺйӮ„жҳҜеҸӘжңүжҺЎз”ЁеҸҚеҚ·ж©ҹйҖІиЎҢ upsampledпјҹFCN и«–ж–Үең–дёӯжһ¶ж§ӢжҳҜеҹәж–ј VGG16 е—Һпјҹ

    • U-NetиҲҮFCNе№ҫд№ҺйғҪжҳҜеҗҢдёҖжҷӮй–“еҮәзҸҫгҖӮжғіе•ҸиҖҒеё«U-NetжҳҜеҸ—еҲ°FCNе•ҹзҷјиҖҢеҮәзҸҫзҡ„жһ¶ж§ӢпјҢйӮ„жҳҜе…©иҖ…еүӣеҘҪйғҪжҳҜеңЁеҗҢдёҖе№ҙеҸ—еҲ°е…ҲеүҚзҡ„еҸҚеҚ·з©Қе•ҹзҷје‘ўгҖӮ

    • Mask R-CNN дёҖй–Ӣе§Ӣзҡ„ Feature extractor жҺЎз”Ё RestNet жҳҜ pretrain еҘҪзҡ„жЁЎеһӢе—ҺпјҹзңҹеҜҰеңЁиЁ“з·ҙжҷӮиҰҒжҠҠ trainable жү“й–ӢжҲ–жҳҜеҸӘй–ӢеҫҢйқўе№ҫеұӨе‘ўпјҹ

    • RoIAlign з°Ўе ұдёӯзҡ„дҫӢеӯҗеҒҮиЁӯеңЁ 2*2 жғ…жіҒдёӢдёҖеҖӢ cell жңүеӣӣеҖӢйҖҸйҒҺйӣҷз·ҡжҖ§е…§е·®жі•еҫ—еҲ°зҡ„й»һгҖӮжҺҘи‘—еҶҚйҖҸйҒҺ max pooling дҝқз•ҷдёҖеҖӢй»һе—Һпјҹжғіи«Ӣж•ҷ RoIAlign жңҖеҫҢеҰӮдҪ•жҺЎз”Ё max pooling дҝқз•ҷе…·жңүд»ЈиЎЁжҖ§зҡ„й»һпјҢжҳҜеҗҰиғҪиҲүз°Ўе–®дҫӢеӯҗгҖӮ

    • FCN и·ҹ U-net еңЁ real time дёҠзҡ„ж•ҲиғҪжҜ”ијғгҖӮ

    • Mask R-CNN ж•ҲиғҪеӨ§зҙ„ 5fps и«Ӣе•ҸжҳҜеңЁд»ҖйәјиҰҸж јдёӢе‘ўпјҹеғҸжҳҜеҪұеғҸеӨ§е°ҸгҖҒGPU е№ҫйЎҶгҖӮ
